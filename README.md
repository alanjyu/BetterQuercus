# Better Quercus

## Installation

- Install a hosting extension for your browser (e.g., [Tampermonkey](https://www.tampermonkey.net/) or [Violetmonkey](https://violentmonkey.github.io/get-it/))
- Click the script you want to install, then click "Raw"
